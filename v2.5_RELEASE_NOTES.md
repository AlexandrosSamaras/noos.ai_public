# Noos.AI v2.5 Release Notes

## 🚀 Gemini 3.1 Migration
We've upgraded our entire core intelligence engine to the latest Gemini 3.1 models. 

- **Faster Summaries:** Enjoy near-instant results for summaries, explanations, and rewrites.
- **Improved Accuracy:** More precise sentiment analysis and keyword extraction.
- **Better Stability:** Fully compatible with the latest Google AI architecture.

## 🎨 CSS Isolation Shield
We've implemented a robust new isolation system to ensure NoosAI looks beautiful on every website.

- **Standardized Fonts:** Fixed issues where some websites (like Guru3D) would force "blocky" or overly bold fonts into the panel.
- **Color Protection:** Improved visibility on dark-mode sites like Facebook, ensuring text is always crisp and readable.
- **Bullet Point Fixes:** Unified bullet point styles across all tech blogs and news sites (no more random green bullets!).

## 🛠️ Performance & Reliability
- Updated backend to support both US and Europe regions for lower latency globally.
- Improved markdown rendering for more consistent formatting across all actions.
